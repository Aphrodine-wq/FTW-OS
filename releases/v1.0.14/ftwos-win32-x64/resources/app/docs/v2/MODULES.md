# FTWOS Modules

## 1. Core
*   **Dashboard**: Central hub with widgets.
*   **Settings**: System configuration and Vault management.
*   **Theme Store**: Advanced customization engine (Glass/Monochrome).

## 2. Finance
*   **Invoices**: Builder, Preview, Print, and History.
*   **Expenses**: Track business costs.
*   **Products**: Manage service catalog.

## 3. CRM
*   **Clients**: Address book and contact details.
*   **Pipeline**: Kanban board for leads.

## 4. Productivity
*   **Tasks**: Todo list with priority.
*   **Time Tracker**: Billable hours tracker with timesheet view.
*   **Documents (Drive)**: File explorer for local documents.

## 5. Office Suite
*   **Mail**: Full IMAP/SMTP client.
*   **Calendar**: Day/Week/Month scheduler.

## 6. DevHQ
*   **Dashboard**: GitHub stats and activity feed.
*   **Terminal**: Integrated system shell.

## 7. Marketing (New)
*   **Dashboard**: Social metrics and growth tracking.
